# Tareas de JS en POO

Este repositorio reúne una colección de prácticas, tareas y ejercicios desarrollados en clase utilizando JavaScript con enfoque en Programación Orientada a Objetos (POO) y estructuras de datos.

## Propósito del Repositorio

El objetivo principal es ofrecer un espacio organizado con ejercicios prácticos que permiten:

- Comprender el funcionamiento de estructuras de datos fundamentales
- Fortalecer la lógica de programación mediante ejemplos reales
- Aplicar conceptos de Programación Orientada a Objetos en JavaScript
- Servir como guía de consulta para tareas y estudio independiente
- Proveer código de referencia para futuros proyectos

## Tecnologías Utilizadas

- JavaScript ES6+
- Programación Orientada a Objetos
- HTML5/CSS3 (para visualizaciones)
- Node.js (para ejecución local)

## Contenido Detallado

### Estructuras de Datos

#### Listas
- Listas Simplemente Enlazadas: Inserción, eliminación, búsqueda y recorrido
- Listas Doblemente Enlazadas: Operaciones bidireccionales
- Métodos Implementados: 
  - agregar(), eliminar(), buscar()
  - recorrer(), obtenerTamaño()
  - estaVacia(), invertir()

#### Pilas (LIFO)
- Implementación con arrays y clases
- Operaciones: push(), pop(), peek(), isEmpty()
- Casos de uso: validación de paréntesis, historial de navegación

#### Colas (FIFO)
- Colas simples y circulares
- Simulaciones de atención al cliente
- Métodos: enqueue(), dequeue(), front()

#### Torre de Hanoi
- Solución recursiva e iterativa
- Visualización del proceso
- Optimización de movimientos

### Programación Orientada a Objetos

#### Conceptos Fundamentales
- Clases y objetos
- Herencia y polimorfismo
- Encapsulamiento y abstracción
- Métodos estáticos y de instancia

#### Proyectos Prácticos
- Sistema de cajero automático
- Gestión de inventarios
- Simulador de procesos
